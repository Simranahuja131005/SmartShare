# SmartShare 📤
A LAN-based File Transfer System built with Java & PostgreSQL.

## Features
- Login & Signup with SHA-256 password hashing
- Send files over LAN using TCP sockets
- Drag & drop file selection
- Real-time transfer progress bar
- File transfer history with timestamps
- Multi-threaded server (handles multiple clients)
- Modern purple-themed Swing GUI

## Tech Stack
- Java (JDK 26)
- Java Swing (GUI)
- TCP Sockets
- PostgreSQL 18
- SHA-256 Security

## How to Run
1. Run `FileServer.java` first
2. Run `LoginUI.java`
3. Sign up, login and start sharing files!
